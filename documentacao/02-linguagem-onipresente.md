# Linguagem Onipresente

Aluno: participante do simulado  
Professor: responsável pelas notas  
Turma: grupo de alunos  
Simulado: avaliação  
Disciplina: matéria  
Nota: resultado  
Ranking: classificação  

Regras:
- Simulado deve ter disciplinas
- Disciplinas possuem peso
- Média é ponderada
- Ranking é ordenado
