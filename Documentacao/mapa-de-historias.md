# Mapa de histórias do usuário

O sistema AcadTrack foi estruturado considerando a jornada dos seus principais atores: coordenador, professor e aluno.

---

## Jornada do Coordenador

- Como coordenador, quero iniciar o processo cadastrando um simulado para organizar uma nova avaliação
- Como coordenador, quero definir disciplinas e pesos do simulado para estruturar corretamente a avaliação
- Como coordenador, quero ajustar a estrutura do simulado antes do encerramento para corrigir possíveis erros
- Como coordenador, quero encerrar o simulado para garantir que não haja alterações após sua finalização

- Como coordenador, quero cadastrar turmas para organizar os alunos de forma estruturada
- Como coordenador, quero vincular alunos às turmas evitando duplicidade de registros
- Como coordenador, quero remover alunos de turmas mantendo a consistência das informações
- Como coordenador, quero transferir alunos entre turmas preservando seu histórico acadêmico

---

## Jornada do Professor

- Como professor, quero lançar as notas dos alunos por disciplina para registrar o desempenho
- Como professor, quero revisar e corrigir notas para garantir a precisão das informações antes da análise final

---

## Jornada do Aluno

- Como aluno, quero acessar minhas notas para acompanhar meu desempenho
- Como aluno, quero visualizar minha posição no ranking para entender minha performance em relação aos demais
