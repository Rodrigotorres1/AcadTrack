# Mapa de histórias do usuário

## Gestão de simulados
- Como coordenador, quero cadastrar um simulado para organizar avaliações
- Como coordenador, quero adicionar disciplinas e pesos ao simulado para definir sua composição
- Como coordenador, quero editar a estrutura de um simulado antes de seu encerramento
- Como coordenador, quero encerrar um simulado para impedir alterações indevidas

## Gestão de turmas e alunos
- Como coordenador, quero cadastrar turmas para organizar os alunos
- Como coordenador, quero vincular alunos a turmas sem duplicidade
- Como coordenador, quero remover alunos de turmas mantendo consistência dos dados
- Como coordenador, quero transferir alunos entre turmas preservando seu histórico

## Avaliação e desempenho
- Como professor, quero lançar notas por disciplina para registrar o desempenho dos alunos
- Como sistema, quero recalcular a média final ponderada automaticamente após o lançamento de notas
- Como coordenador, quero gerar o ranking dos alunos com base no desempenho
- Como sistema, quero aplicar critério de desempate no ranking para manter a ordenação consistente
