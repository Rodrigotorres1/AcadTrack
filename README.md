# AcadTrack

Plataforma acadêmica de gestão de simulados.

## Objetivo
Permitir o cadastro de simulados, disciplinas, turmas, alunos, notas e rankings.

## Estrutura
- dominio-principal: regras de negócio
- aplicacao: casos de uso e DTOs
- infraestrutura: persistência
- apresentacao-backend: API
- apresentacao-frontend: protótipos da interface
- Documentacao: artefatos da 1ª entrega

## Linguagem Onipresente
Aluno, Simulado, Disciplina, Nota, Ranking
